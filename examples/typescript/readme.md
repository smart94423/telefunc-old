We share the backend types with the frontend by using Telefunc's [TypeScript support](/../../#typescript).

To run the example:

1. Clean caches.
  ~~~bash
  yarn clean
  ~~~

2. Build everything and start the server.
  ~~~bash
  yarn start
  ~~~

3. Go to [http://localhost:3000](http://localhost:3000).
